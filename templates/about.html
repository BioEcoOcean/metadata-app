About

These tools are being developed by Elizabeth Lawrence from OBIS, produced as part of outcomes from the BioEcoOcean Horizon Europe project. AI (ChatGPT-4 and -5) was used to assist with the development and all code generated was reviewed, tested, and edited as necessary before being implemented.

Metadata about data producers ultimately should be visible in ODIS and the GOOS BioEco Portal. BioEco EOV data should be published to OBIS, which requires it to be formatted according to the Darwin Core standard. This tool is designed to facilitate the process of creating metadata and data files that are compliant with these standards.

See the following GitHub repos for details about the app development and where submitted entries are hosted on GitHub.
