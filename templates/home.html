Submit EOV Metadata

This application is designed to streamline the creation of metadata JSON-LD files and sitemaps for Essential Ocean Variables (EOV) "data producers", e.g. programmes, projects, and/or institutions. This tool facilitates seamless integration with the GOOS BioEco Portal and the Ocean Data and Information System (ODIS). Generated JSON-LD files are submitted to the BioEcoOcean GitHub repository as issues, when entries are updated, a date is added in a comment.

You must have a GitHub account to use this tool as submissions are sent to the GitHub repository.
