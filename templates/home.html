Submit EOV Programme Metadata

This application is designed to streamline the creation of metadata JSON-LD files and sitemaps for Essential Ocean Variables (EOV) programmes. It facilitates seamless integration with the GOOS BioEco Portal and the Ocean Data and Information System (ODIS). Generated JSON-LD files are submitted to the BioEcoOcean GitHub repository as issues, with updates tracked as comments.

You must have a GitHub account to use this tool as submissions are sent to the GitHub repository.
