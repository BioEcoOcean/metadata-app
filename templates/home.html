Submit EOV Metadata

Fill in the form to provide metadata describing any organized monitoring effort for Essential Ocean Variables (EOV). We call these monitoring efforts "data producers", which can include monitoring programmes, projects, institutions, etc. Submitting an entry via this application will allow it become visible in both the GOOS BioEco Portal and the Ocean Data and Information System (ODIS) / Ocean Info Hub. Note that as of August 2025 the connection with these is still a work in progress, as the BioEco Portal is being redesigned. You may still submit entries via this tool.

You must have a GitHub account to use this tool as submissions are sent to the GitHub repository. Your Github account is displayed below if you are properly logged in. Click the "Add metadata" button below to begin submitting metadata.
